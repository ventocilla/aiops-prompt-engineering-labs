# Providers and Models Used

This repository uses multiple AI providers and models in order to compare:
- output quality
- reasoning capability
- adherence to prompt structure
- infrastructure/code generation quality
- operational analysis capability

## Providers

| Provider | Models Used | Typical Use Cases |
|---|---|---|
| OpenAI | ChatGPT 5.5 / GPT-4.1 equivalent | Docker, Terraform, SQL, Kubernetes |
| Anthropic | Claude Sonnet 4 | Operational runbooks, Bash scripting, YAML |
| Google | Gemini 2.5 Pro | Incident analysis, structured reasoning |

## Selection Criteria

Models were selected based on:
- code generation quality
- structured output consistency
- reasoning depth
- operational context understanding
- infrastructure and SRE knowledge

## Notes

The same framework may behave differently depending on the provider and model used.

Part of the objective of this repository is to evaluate these differences critically.
